img태그

img태그가 들어가면 그 의미에 맞는 alt값을 넣어준다

IR기법(Image Replacement) - 이미지 대체기법 IR기법으로 대체할수 있다.
IR기법

배경 이미지의 의미가 사용자에게 전달되어야 하는 콘텐츠는 그 의미가 보조 기술로 전달되도록 대체 텍스트를 제공해야 한다. 

                        
페이스북으로 이동

                    
						/* bg 로 이미지 삽입*/
                        .btn_facebook{display:inlnie-block;width:20px;height:20px;text-indent:-999em;font-size:0;backgroung:url(/images/btn_facebook.png) no-repeat;}
title 태그

페이지에 제목이라 할 수 있는 <title>태그에는 간략하게 페이지 별로 페이지의 대한 정보(제목)만을 입력한다.

무의미한 기호나 텍스트는 넣지 않는다.

                        공지사항 :: 씨엔티테크
Skip Navigation 제공

장애인을 위해 리더기가 내용을 건너 뛸 수 있게 skip navigation을 제공한다.

skip navigation은 장애인을 위한 내용이기에 화면상에 보이지 않게 처리해도 무방하다.

table 태그

Table 태그에는 <summary> 속성 (테이블을 간단히 요약한 내용) 값을 넣어준다.

Table 태그 안에는 <caption>태그를 넣어 테이블의 제목값을 넣어준다.

<thead>나 <tbody>로 나눠서 의미에 맞게 사용하고 <tfoot>은 바닥글이 있을때만 사용한다. 테이블이 길 경우 리더기가 얼마나 오래 읽어야 할지 모르는 불편함을 덜기 위해 작성순서를 thead – tfoot – tboody 순으로 한다

<th> 에는 <scope> 속성을 넣거나 id 값을 지정하여 <td>에 <headers> 속성으로 연결한다.

<th>에 <abbr>속성을 이용하여 약어도 넣어줄 수 있다.

input 태그

<input>태그,<textarea>태그에는 title속성을 이용하여 역할이나 내용을 입력한다.

<label>태그를 같이 쓸 경우 <input>태그에 id값을 지정하여 <label>태그의 for속성을 이용하여 연동 시켜준다.

<label> 들어갈 공간이 협소할때 <title>로 제공하여도 접근성에 위배되지 않는다.
